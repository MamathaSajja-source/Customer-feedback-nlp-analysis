# Customer-feedback-nlp-analysis
AI-powered NLP system for customer feedback analysis
## Overview 

This project analyzes customer reviews and converts them into business insights using NLP.

## Features

- Sentiment analysis
- Identifies key issues
- Generates business insights

## Results

- 85% positive reviews
- Major complaints: Quality, battery, charging

## Tech Stack

- Python
- NLP
- Pandas
